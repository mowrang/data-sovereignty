## Step-by-Step Signup Guides

### Adzuna (Easiest - Start Here)

1. Go to: https://developer.adzuna.com/overview
2. Click "Get Started" or "Sign Up"
3. Enter email and create password
4. Verify email
5. Get your App ID and API Key
6. Start using API immediately!

**Time:** 5 minutes  
**Cost:** Free  
**API Access:** Instant

**Add to .env:**

```bash
ADZUNA_APP_ID=your_app_id
ADZUNA_API_KEY=your_api_key
ADZUNA_REVENUE_PER_CLICK=0.15
ADZUNA_ENABLED=true
```

---

### Indeed (Highest Revenue)

1. Go to: https://www.indeed.com/publisher
2. Click "Apply Now" or "Get Started"
3. Fill out application form:
   - Website URL
   - Traffic estimates
   - How you'll use jobs
   - Business information
4. Submit application
5. Wait for approval (1-2 weeks)
6. Get Publisher ID and API Key
7. Set up affiliate links

**Time:** 15 minutes (application) + 1-2 weeks (approval)  
**Cost:** Free  
**API Access:** After approval

**Add to .env:**

```bash
INDEED_PUBLISHER_ID=your_publisher_id
INDEED_API_KEY=your_api_key
INDEED_AFFILIATE_ID=your_affiliate_id
INDEED_REVENUE_PER_CLICK=0.25
INDEED_ENABLED=true
```

---

### ZipRecruiter (Very High Revenue)

1. Go to: https://www.ziprecruiter.com/affiliates
2. Click "Join Now" or "Apply"
3. Fill out affiliate application:
   - Website information
   - Traffic data
   - Marketing plan
4. Submit application
5. Wait for approval
6. Get API access and affiliate links

**Time:** 20 minutes (application) + approval time  
**Cost:** Free  
**API Access:** After approval

**Revenue:** Highest rates ($0.50-$1.00 per click)
